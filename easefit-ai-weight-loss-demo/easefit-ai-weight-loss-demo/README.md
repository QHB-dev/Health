# EaseFit AI Weight Loss Demo

这是一个可部署的 React + Vite + Tailwind CSS 网页 Demo，用于展示 EaseFit AI 陪伴式减重 App 原型。

## Core Modules

- Lightweight Onboarding
- Daily Micro Plan
- AI Reset Coach
- AI Vision Movement Guidance
- Progress Beyond Weight
- Health Records
- Stage Upgrade / 老手模式
- Light Social
- Premium Module

## How to Run Locally

```bash
npm install
npm run dev
```

## How to Build

```bash
npm run build
```

## Deploy on Vercel

1. 将整个项目上传到 GitHub Repo。
2. 打开 Vercel，用 GitHub 登录。
3. Add New Project → Import Git Repository。
4. 选择本项目，保持默认配置：
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
5. 点击 Deploy。

部署成功后，Vercel 会生成可访问的 Live Demo 链接。
